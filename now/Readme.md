# /now

This directory contains the only active writing on the site.

## What qualifies
- Must respond to something external
- Must take a clear position
- 300â€“800 words (150 minimum if complete)
- Written in one sitting, edited once
- Humane, non-performative tone

## Required structure
- Name the trigger
- Respond
- State what changes if true

## What does not belong here
- Drafts
- Notes
- Teaching pieces
- Announcements
- Sales material
- Multi-part work

If a text does not meet these criteria, it does not belong in /now.
